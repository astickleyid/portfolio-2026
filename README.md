# austin stickley — portfolio

Personal portfolio. Five live products, designed and engineered by one operator.

## Stack

Single static `index.html`. Inter via Google Fonts. Lenis smooth scroll. No build step.

## SEO & marketing

- Full Open Graph + Twitter card meta
- JSON-LD structured data (Person, WebSite, ItemList of products)
- `og.jpg` rendered at 1200x630
- `sitemap.xml` + `robots.txt`
- Vercel Analytics + Speed Insights (auto-active on deploy)
- Mobile-first responsive, semantic HTML, a11y skip link

## Local

Open `index.html` in any browser.

## Deploy

Auto-deploys on push to `main` via Vercel.
